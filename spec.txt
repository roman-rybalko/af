ТЗ:

Имеется макет на основе xslt.

Требуется 3 сайта: полноформатный, мобильный, SEO для яндекса (яндекс - приоритет).
Контент должен находиться в подкаталоге /static.
/static/main - полноформатный
/static/mobile - мобильный
/static/seo - оптимизированный для поисковиков
Корневой файл шаблона должен называться default.xsl (/static/main/default.xsl, /static/mobile/default.xsl, /static/seo/default.xsl).
Изменять входные данные можно только в каталоге /static.
xml-файлы даны как пример, содержимое будет изменяться, структура и набор тегов сохранится.

Требуется:
Простой уникальный дизайн, цвета: синий, голубой, бирюзовый.
Создать диаграммы по макетам (2 диаграммы), встроить в общий дизайн.
Название оргинизации "ПРОГРЕССИВНАЯ ФИЛЬТРАЦИЯ" (желательно крупный шрифт с наклоном и цветовым градиентом), встроить в общий дизайн.
Меню - картинки или стили. Внимание: имеется основное меню и меню личного кабинета. Активные элементы меню должны переставляться без искажения дизайна.
Дизайн для элементов управления, пример в разделе "личный кабинет" макета.
